## 如何使用
* 安装依赖 pip3 install python-telegram-bot
* 编辑config.py,修改为您的配置
* 使用如下命令运行:python tg2qq.py

* [安装coolq](https://cqp.cc/)
* [安装cqhttp插件](https://github.com/richardchien/coolq-http-api/releases)
* [打开websocket功能](https://cqhttp.cc/docs/4.15/#/WebSocketAPI)
